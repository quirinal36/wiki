# Start Here

This vault is the working home for the LLM Wiki, Obsidian workflow, and Hermes-managed research notes.

## What this vault is for
- Build a persistent markdown knowledge base instead of re-deriving the same knowledge repeatedly.
- Keep source material in `raw/` and reusable knowledge in `wiki/`.
- Use Obsidian as the reading and navigation interface.
- Use Hermes as the disciplined maintainer of the wiki.

## Operating flow
1. Save source material into `raw/`.
2. Summarize and connect insights in `wiki/`.
3. Update [[index]] and `log.md` whenever meaningful work is done.
4. Prefer updating an existing page instead of creating a duplicate.
