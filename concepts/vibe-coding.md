# Vibe Coding

## Definition
AI(특히 LLM)의 지원을 받아 전통적인 코딩 문법 숙달보다 전체적인 로직의 흐름과 '바이브(느낌/의도)'를 제어하여 빠르게 결과물을 만들어내는 개발 방식.

## Context
전통적인 개발자 교육이 '컴파일러 언어의 문법'과 '알고리즘'에 집중했다면, Vibe Coding은 '프롬프트 엔지니어링', '아키텍처 설계', 'AI와의 협업'에 집중한다.
