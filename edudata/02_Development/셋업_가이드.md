# 🛠️ 개발 셋업 가이드 (For 류승엽, 김우진)

이 문서는 프로젝트에 새로 참여한 팀원을 위한 가이드입니다.

## 1. 환경 준비
- **Node.js**: v18.0.0 이상 권장
- **Package Manager**: `pnpm` (권장) 또는 `npm`
- **Editor**: VS Code (추천 플러그인: Tailwind CSS IntelliSense, ESLint)

## 2. 저장소 클론 및 설치
```bash
git clone https://github.com/SeungYeoop/yh_meal.git # (현재 edudata 전용으로 전환 예정)
cd edudata
pnpm install
```

## 3. 환경 변수 설정 (`.env.local`)
다음 값이 필요합니다 (Buddy에게 요청하세요).
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `ANTHROPIC_API_KEY`

## 4. 로컬 실행
```bash
pnpm dev
```
