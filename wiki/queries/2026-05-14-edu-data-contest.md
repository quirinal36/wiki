# [교육공공데이터 AI 활용대회] 청소년 진로 탐색 프로젝트 기획안

> **Goal:** 교육공공데이터와 AI를 결합하여 청소년들이 자신의 적성과 진로를 주도적으로 찾아갈 수 있도록 돕는 서비스 구축

## 1. 프로젝트 개요
- **대회 명칭:** 제8회 교육공공데이터 AI 활용대회
- **핵심 컨셉:** "데이터로 그리는 나의 미래" - 파편화된 교육 데이터를 통합하여 개인화된 진로 로드맵 제시
- **타겟:** 진로 선택에 어려움을 겪는 중·고등학생 및 학부모

## 2. 주요 기능 (아이디어)
1. **나의 특성 데이터 분석:** (가상/실제) 성적, 관심사, 활동 기록 입력을 통한 프로파일링
2. **진로 매칭 엔진:** 
   - 커리어넷 직업/학과 데이터와 공공데이터 기반 유망 직종 매칭
   - AI(LLM) 기반 맞춤형 진로 상담 및 로드맵 생성
3. **진로 인프라 시각화:** 
   - 학교 알리미 및 전국 진로체험처 데이터를 연동한 내 주변 교육 자원 지도 서비스
4. **유사 선배 경로 추천:** 공공데이터 상의 학과 진학 통계 기반 성공 경로 제시

## 3. 기술 스택
- **언어/프레임워크:** Python (Data Processing), Next.js/React (Frontend)
- **데이터베이스:** Supabase (Auth, DB, Storage)
- **AI/ML:** OpenAI api or Claude (진로 컨설팅)
- **데이터 소스:** 교육데이터 서비스(EDSS), 커리어넷 API, 공공데이터포털 등

## 4. 향후 추진 계획
- **Phase 1:** 데이터 소스 확정 및 API 연동 테스트
- **Phase 2:** AI 진로 매칭 알고리즘 설계 및 프로토타입 개발
- **Phase 3:** 사용자 반응 조사 및 UI/UX 고도화
- **Phase 4:** 대회 제출용 영상 및 기획서 제작
