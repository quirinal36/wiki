# Trader Agent

## Overview
Trader Agent is a focused operating profile for trading-related workflows, analysis requests, and gateway-based automation.

## Why it matters
- It provides a dedicated operating context separate from the default profile.
- It can be managed as a persistent Hermes gateway workflow.
- It gives a clear place to document trading processes, assumptions, and future operating rules.

## Current context
- The trader profile is configured as a Hermes gateway.
- It is now set up to auto-start after Windows login through WSL and systemd.
