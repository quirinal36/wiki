## Overview
Design requirements for a web-based control plane for [[hermes-agent]]. Focuses on managing agents, teams, tasks, and handoffs rather than simple chat. Prefers a decoupling of the Control Plane and the Web UI.

## Features
- Visual team management.
- Task and run status monitoring.
- Handoff visualization.
